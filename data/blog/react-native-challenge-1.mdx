### JSX

- 실제로 우리가 작성한 JavaScript 코드가 실행되는 영역과, UI를 그려주는 네이티브 영역은 분리되어 있다.
- How? (RN 에서 )
- React Native Bridge 매커니즘을 통해 JSX와 네이티브 실행 영역에서 서로 소통할 수 있도록 통로가 있다.
- Brige 매커니즘: 분리된 영역에서 서로 메세지-큐 방식으로 소통할 수 있는 매커니즘
- yoga: meta에서 flexbox를 보여주기 위해 만든 크로스 플랫폼 앤진
- flexbox 레이아웃 엔진으로 다양한 프로그래밍 언어에서 사용할 수 있는 오픈소스 라이브러리
- flexbox는 web 에서만 존재하고, 네이티브 모바일에서는 존재 하지 않음
